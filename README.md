## 📋 Ejercicios Completados

### Ejercicios de Lógica — Pseudocódigo sin Computador
Conjunto de ejercicios prácticos para desarrollar habilidades de lógica y pseudocódigo.

#### ✅ Ejercicio A — Clasificar una contraseña
Clasificación de contraseñas según su fortaleza:
- **Débil:** menos de 8 caracteres
- **Media:** 8+ caracteres con solo letras/números
- **Fuerte:** 8+ caracteres con letras, números y símbolos
- **Sin contraseña:** campo vacío

**Estado:** Completado ✓

#### ✅ Ejercicio B — Verificar nodos del cluster
Verificación de disponibilidad de nodos en un cluster de red:
- Hacer ping a cada nodo de la lista
- Clasificar como "activos" o "caídos"
- Mostrar estadísticas de disponibilidad
- Considerar casos especiales (lista vacía, ninguno responde)

**Estado:** Completado ✓

#### ✅ Ejercicio C — Validar un email básico
Validación de formato de correo electrónico con reglas específicas:
- Exactamente un símbolo @
- El @ no puede ser primer ni último carácter
- Debe haber al menos un punto después del @
- No puede haber espacios

**Estado:** Completado ✓

#### ✅ Ejercicio D — Contar palabras únicas
Análisis de texto para contar palabras distintas:
- Input: Texto con palabras (posiblemente repetidas)
- Output: Número de palabras únicas
- Ejemplo: "el gato y el perro y el gato" → 4 palabras (el, gato, y, perro)

**Estado:** Completado ✓
